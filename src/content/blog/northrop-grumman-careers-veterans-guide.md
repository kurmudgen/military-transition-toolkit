## Northrop Grumman Hiring Process

### Step 1: Apply

**Where to apply:**
- careers.northropgrumman.com
- ClearanceJobs
- LinkedIn
- Indeed

**Requirements:**
- Resume upload
- Application questions
- Clearance status
- Veteran status

### Step 2: Screening

HR review for:
- Basic qualifications
- Clearance eligibility
- Experience match
- Education verification

### Step 3: Interviews

Multiple rounds typical:

**Phone Screen:**
- Initial discussion
- Background review
- Interest confirmation

**Technical Interview:**
- Role-specific questions
- Skills assessment
- Problem-solving

**Panel/Manager Interview:**
- Team fit
- Behavioral questions
- Leadership assessment

### Step 4: Assessment

Some positions include:
- Technical assessment
- Skills testing
- Security interview
- Background check

### Step 5: Offer

If selected:
- Compensation discussion
- Benefits review
- Start date coordination
- Clearance processing

### Timeline

Typical process:

- Interview process: 2-6 weeks
- Offer to start: 2-4 weeks

Clearance processing adds time if needed.
