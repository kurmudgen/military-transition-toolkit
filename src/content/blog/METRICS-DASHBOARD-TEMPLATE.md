## HOW TO USE THIS DASHBOARD

**Weekly** (Every Friday, 30 minutes):
1. Fill out Sections 1-10 (Traffic through Social Media)
2. Review "Weekly Insights" - what worked, what didn't?
3. Set Top 3 Priorities for next week
4. Share with team (if applicable) or review solo

**Monthly** (First Monday, 2 hours):
1. Complete Monthly Deep Dive sections
2. Analyze cohorts, channels, content, features
3. Make strategic decisions (scale, optimize, cut)
4. Set next month's goals and resource allocation
5. Document and review progress toward 90-day goals

**Quarterly** (Every 3 months, 4 hours):
1. Review all 3 months of data
2. Identify trends and patterns
3. Major strategic pivots if needed
4. Set next quarter's OKRs (Objectives & Key Results)
5. Celebrate wins!
