## Keyword Research Methodology

### Metrics Explained

**Search Volume:**
- Monthly average searches (US)
- Higher = more traffic potential

**Keyword Difficulty (KD):**
- 0-20: Easy (low competition)
- 21-40: Medium (achievable)
- 41-60: Hard (competitive)
- 61-100: Very Hard (requires authority)

**Search Intent:**
- **Informational:** User wants to learn
- **Navigational:** User wants specific site
- **Transactional:** User wants to take action
- **Commercial Investigation:** User researching before decision

**CPC (Cost Per Click):**
- How much advertisers pay
- Indicates commercial value
- Higher = more valuable keyword

## Competitor Analysis (Top Rankings to Beat)

### For Each Top 100 Keyword, Identify:

**1. Top 3 Ranking Pages:**
- What's their word count?
- What sections do they cover?
- What's missing (opportunity)?
- What's their domain authority?

**2. SERP Features Present:**
- Featured snippet (what format?)
- People Also Ask questions
- Videos
- Images
- Local pack

**3. Content Gaps:**
- What questions aren't answered?
- What's outdated (older data)?
- What's missing (calculations, tools)?

**Example: "VA Disability Rates 2025"**

**Top Ranking Sites:**
1. VA.gov (official, DA 95)
2. Military.com (DA 82)
3. Veterans United (DA 68)

**Content Gaps:**
- None have interactive calculators
- None compare 2024 vs. 2025 rates clearly
- None have downloadable PDF chart
- None explain VA math in simple terms

**Our Advantage:**
- Add calculator tool
- Add comparison table
- Offer PDF download
- Simplify VA math explanation
- Update monthly (keep fresh)

## Keyword Research Tools Workflow

### Free Tools
1. **Google Keyword Planner**
   - Get search volume estimates
   - Find related keywords

2. **AnswerThePublic**
   - Find question-based keywords
   - Discover "people also ask" variations

3. **Google Search Console**
   - See what you already rank for
   - Find keyword opportunities (position 11-20)

4. **Google Autocomplete**
   - Type "[keyword]" and see suggestions
   - Type "[keyword] for" / "[keyword] to" / "[keyword] vs"

5. **AlsoAsked.com**
   - See "People Also Ask" questions

1. **Ahrefs** ($99/mo)
   - Keyword difficulty

2. **Semrush** ($119/mo)
