## SEO Optimization Framework

### Keyword Research & Targeting

#### Primary Keywords (High Volume, High Intent)
Focus: 500+ searches/month, commercial intent

**Examples**:
- "military to civilian career transition" (5,400/month)
- "veteran career resources" (1,900/month)
- "[MOS] to civilian jobs" (varies: 100-2,400/month)
- "VA disability rating for [condition]" (varies: 1,000-8,000/month)
- "[state] veteran benefits" (varies: 200-2,000/month)

**Optimization**:
- Primary keyword in title (front-loaded)
- Primary keyword in URL slug
- Primary keyword in H1
- Primary keyword in first 100 words
- Primary keyword density: 0.5-1%

#### Secondary Keywords (Related Terms)
**Examples**:
- Primary: "11B infantry to civilian career"
- Secondary: "infantry jobs after military", "what jobs can infantry get", "infantry skills in civilian world"

**Optimization**:
- Use in H2/H3 subheadings
- Natural integration in body content
- Include in meta description
- Use in image alt tags

#### Long-Tail Keywords (Low Competition, High Intent)
**Examples**:
- "Can a 68W become a paramedic"
- "Do Texas veterans pay property tax"
- "PTSD secondary to TBI VA rating"

**Optimization**:
- Target in FAQ sections
- Use conversational language
- Optimize for voice search
- Answer specific questions directly

### On-Page SEO Checklist

For every published post, ensure:

#### Title Tag (55-60 characters)
- [ ] Primary keyword included
- [ ] Front-loaded (keyword first)
- [ ] Compelling and click-worthy
- [ ] Year included for freshness (2025)
- [ ] Example: "11B Infantry to Civilian Career Guide (2025 Updated)"

#### Meta Description (150-160 characters)
- [ ] Primary keyword included
- [ ] Secondary keyword included
- [ ] Clear value proposition
- [ ] Call to action
- [ ] Example: "Complete guide to transitioning from 11B Infantry to civilian careers. Discover top jobs, salary ranges, certifications needed, and companies hiring. Free career tools inside."

#### URL Structure
- [ ] Short and descriptive
- [ ] Primary keyword included
- [ ] Hyphens for word separation
- [ ] No stop words (a, the, and, etc.)
- [ ] Example: `/11b-infantry-civilian-career-guide/`

- [ ] One H1 (title) with primary keyword
- [ ] H2s with secondary keywords

- [ ] Logical hierarchy

## Featured Snippets & PAA Targeting

### What Are Featured Snippets?
"Position Zero" results that appear above organic listings. Highly valuable:
- 35% CTR (vs 26% for #1 organic result)
- Voice search optimization
- Authority signal

### Types of Featured Snippets

#### 1. Paragraph Snippets (Most Common)
**Query**: "What is the 11B MOS in the Army?"

**Optimization**:
- Create H2 that mirrors the question
- Answer in 40-60 words immediately after heading
- Use clear, concise language
- Include the question in the answer

**Example**:
```markdown
## What Is the 11B MOS in the Army?

The 11B MOS (Military Occupational Specialty) is the Army's Infantry
designation. 11B soldiers are trained as riflemen and serve as the
primary combat force in ground operations. They engage enemy forces,
capture territory, and defend positions using individual weapons and
small unit tactics.
```

#### 2. List Snippets
**Query**: "Best civilian careers for infantry veterans"

**Optimization**:
- Create H2 with question
- Use numbered list or bullet points
- Include 5-10 items
- Keep each item concise (1-2 sentences)

**Example**:
```markdown
## Best Civilian Careers for Infantry Veterans

1. **Law Enforcement Officer** - Police, sheriff, federal agent roles
   leverage tactical skills and discipline
2. **Security Specialist** - Corporate security, executive protection,
   cybersecurity roles
3. **Emergency Responder** - Firefighter, EMT, paramedic positions value
   high-stress experience
4. **Project Manager** - Leadership and operational planning skills translate
   directly
5. **Operations Manager** - Logistics, supply chain, and operations roles
```

#### 3. Table Snippets
**Query**: "VA disability rating percentages"

**Optimization**:
- Use HTML tables or markdown tables
- Include clear headers
- Keep data scannable
- Include source citation

**Example**:
```markdown
## VA Disability Rating Percentages

| Condition | 0% | 10% | 30% | 50% | 70% | 100% |
|-----------|-----|-----|-----|-----|-----|------|
| PTSD | Diagnosed, no symptoms | Mild symptoms | Moderate symptoms | Serious symptoms | Severe symptoms | Total impairment |
| Tinnitus | Not rated | Rated | Not rated | Not rated | Not rated | Not rated |
| Back Pain | Asymptomatic | Limited flexion | Moderate limitation | Severe limitation | Pronounced limitation | Complete disability |
```

#### 4. Video Snippets
**Query**: "How to file a VA disability claim"

**Optimization**:
- Embed YouTube video in blog post
- Use exact match title
- Include timestamp descriptions
- Add video transcript
- Optimize video title, description, tags

### People Also Ask (PAA) Strategy

**What is PAA?**
Google's "People Also Ask" boxes appear for 43% of all queries. Targeting
these increases visibility and drives traffic.

#### PAA Research Process

1. **Identify PAAs for Target Keywords**
   - Google your target keyword
   - Note all PAA questions
   - Click each PAA to reveal more questions
   - Document 10-20 related questions

2. **Create FAQ Sections**
   - Include at minimum 5 FAQs per post
   - Use exact PAA questions as H3 headings
   - Answer concisely (50-100 words per FAQ)
   - Use FAQ schema markup

**Example: 11B Infantry Career Guide PAAs**

```markdown
## Frequently Asked Questions

### What civilian jobs can 11B Infantry get?

11B Infantry veterans qualify for law enforcement, security, emergency
response, project management, and operations roles. Top civilian careers
include police officer, firefighter, security specialist, project manager,
and logistics coordinator. Many employers value infantry leadership,
tactical skills, and discipline.

### How much do infantry veterans make in civilian jobs?

Infantry veterans earn $40,000-$75,000 annually in entry-level civilian
roles, with potential to reach $60,000-$100,000+ with certifications and
experience. Law enforcement averages $55,000-$70,000, security specialists
earn $45,000-$65,000, and project managers make $65,000-$90,000.

### Do infantry veterans need certifications for civilian jobs?

While not always required, certifications significantly boost salary and
hiring prospects. Recommended certifications include: POST (law enforcement),
PMP (project management), Security+, EMT-Basic, and CPR/First Aid. Most
certifications cost $500-$3,000 and take 3-12 months to earn.

### How long does it take for infantry veterans to find civilian jobs?

Most infantry veterans find employment within 3-6 months of active job
searching. Success factors include: starting 12+ months before separation,
earning relevant certifications, networking actively, and using military-
friendly job boards. Veterans who plan ahead typically receive multiple
offers.

### What are the best companies for infantry veterans?

Top employers for infantry veterans include law enforcement agencies (FBI,
DEA, state/local police), defense contractors (Lockheed Martin, Northrop
Grumman), security firms (Allied Universal, G4S), and project-focused
companies (Amazon, FedEx). These employers actively recruit veterans and
offer transition support.
```

## Content Update & Refresh Strategy

### Why Update Content?
- Google favors fresh content
- Improves rankings over time
- Captures new keywords
- Adds new data and insights
- Fixes outdated information

### Update Prioritization

**Update These Posts First**:
1. Currently ranking #2-#10 (easiest to push to #1)
2. High traffic but declining
3. Posts with outdated data (pre-2024)
4. Posts missing schema, internal links, or CTAs
5. Posts with high bounce rate (add more value)

### Update Frequency

**High-Priority Posts** (Top 20 by traffic):
- Review monthly
- Update quarterly
- Add new data, stats, examples
- Refresh publish date

**Medium-Priority Posts** (Ranking #2-#20):
- Review quarterly
- Update every 6 months
- Expand content (add 500+ words)
- Improve internal linking

**Low-Priority Posts** (Long-tail, lower traffic):
- Review annually
- Update as needed
- Maintain accuracy

### What to Update

**Content Improvements**:
- [ ] Add new data and statistics
- [ ] Include recent examples or case studies
- [ ] Expand thin sections (add 200-500 words)
- [ ] Update year references (2024 → 2025)
- [ ] Add new FAQs from search queries
- [ ] Include user-generated content (comments, testimonials)

**SEO Improvements**:
- [ ] Add missing schema markup
- [ ] Improve internal linking (3-5 links minimum)
- [ ] Optimize for new keywords (check Search Console)
- [ ] Add alt text to images
- [ ] Improve title and meta description
- [ ] Optimize for featured snippets

**Conversion Improvements**:
- [ ] Add/improve CTAs
- [ ] Link to new lead magnets
- [ ] Promote new tools
- [ ] Add social proof (testimonials, stats)
- [ ] Reduce friction (simplify CTAs)

**Publish Date Strategy**:
- Update "dateModified" in schema
- Add "Last Updated: [Date]" at top of post
- Refresh publish date in CMS (if significant update)
- Signals freshness to Google

## Content Creation Workflow

### Step-by-Step Process

#### Step 1: Keyword Research (15 minutes)
- [ ] Identify target keyword
- [ ] Check search volume (Ahrefs, Semrush, Google Keyword Planner)
- [ ] Assess competition (keyword difficulty)
- [ ] Identify secondary keywords (2-5)
- [ ] Document PAA questions

#### Step 2: Outline Creation (15 minutes)
- [ ] Create H2/H3 structure
- [ ] Plan sections to cover
- [ ] Identify data/stats needed
- [ ] Plan internal link opportunities
- [ ] Determine CTA placement

#### Step 3: Content Writing (60-90 minutes)
- [ ] Write introduction (hook + value prop)
- [ ] Write body (address each H2/H3)
- [ ] Write FAQ section (5-10 questions)
- [ ] Write conclusion (summary + CTA)
- [ ] Add statistics and data
- [ ] Include examples and stories
- [ ] Aim for 1,500-2,500 words

#### Step 4: SEO Optimization (15 minutes)
- [ ] Optimize title tag
- [ ] Write meta description
- [ ] Set URL slug
- [ ] Add keyword to first 100 words
- [ ] Use keywords in headings
- [ ] Check keyword density (0.5-1%)

#### Step 5: Visual Elements (15 minutes)
- [ ] Create/select featured image
- [ ] Add 2-4 supporting images
- [ ] Create tables or charts if applicable
- [ ] Add alt text to all images
- [ ] Compress images for speed

#### Step 6: Internal Linking (10 minutes)
- [ ] Link to pillar page
- [ ] Link to 2-3 related posts
- [ ] Link to lead magnet
- [ ] Link to free tool
- [ ] Link to conversion page
- [ ] Use descriptive anchor text

#### Step 7: Schema & Technical (10 minutes)
- [ ] Add Article schema
- [ ] Add FAQ schema (if FAQ section)
- [ ] Add How-To schema (if applicable)
- [ ] Test schema in Rich Results Test
- [ ] Set publish date
- [ ] Set categories and tags

#### Step 8: CTAs & Conversion (10 minutes)
- [ ] Add above-fold CTA
- [ ] Add mid-content lead magnet CTA
- [ ] Add end-of-post trial CTA
- [ ] Add sidebar CTAs
- [ ] Test all CTA links

#### Step 9: Quality Check (10 minutes)
- [ ] Read through for clarity
- [ ] Check grammar and spelling (Grammarly)
- [ ] Verify all links work
- [ ] Check images display correctly
- [ ] Preview on mobile
- [ ] Review for accuracy

#### Step 10: Publish & Promote (15 minutes)
- [ ] Publish post
- [ ] Share on LinkedIn
- [ ] Share on Reddit (if relevant)
- [ ] Share in Facebook groups
- [ ] Send to email list (if appropriate)
- [ ] Submit to Google (Search Console)

**Total Time Per Post**: ~3 hours (with practice, reduce to 2 hours)
