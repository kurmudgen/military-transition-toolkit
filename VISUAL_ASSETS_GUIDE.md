**Dimensions:**
- **Standard:** 1270Ã—760px (Product Hunt, most directories)
- **Alternative:** 1200Ã—800px (BetaList, some platforms)

5. Add subtle drop shadows if helpful (optional)
6. Ensure text is readable at thumbnail sizes

## Creating Screenshots - Step by Step

### Option 1: Manual Screenshots (Free)

**For macOS:**
1. Navigate to the page you want to capture
2. Press `Cmd + Shift + 4` then press `Space` to capture window
3. Or use `Cmd + Shift + 5` for screenshot toolbar with more options
4. Click window to capture
5. Screenshot saved to Desktop

**For Windows:**
1. Navigate to the page you want to capture
2. Press `Windows + Shift + S` for Snip & Sketch
3. Select area to capture
4. Edit and save

**For Linux:**
1. Use `gnome-screenshot` or `scrot`
2. Or use Spectacle (KDE), GNOME Screenshot, or Flameshot

**Browser-based (All platforms):**
1. Open browser Developer Tools (F12)
2. Toggle device toolbar (Ctrl+Shift+M / Cmd+Shift+M)
3. Select device or set custom dimensions (1270Ã—760)
4. Use screenshot tool in dev tools menu (three dots â†’ Capture screenshot)

### Option 2: Screenshot Tools (Recommended)

**Free Tools:**
- **Greenshot** (Windows) - Free, open-source
- **Flameshot** (Linux) - Free, feature-rich
- **Firefox Screenshot Tool** - Built into browser
- **Chrome DevTools** - Built into browser

**Paid Tools (Optional):**

## Screenshot Post-Processing

1. **Crop to exact dimensions** (1270Ã—760px)

## Logo Creation Guide

If you need to create or update your logo:

### Logo Design Principles

1. **Simplicity** - Should work at 32Ã—32px
2. **Relevance** - Convey military/transition theme
3. **Memorability** - Unique and recognizable
4. **Versatility** - Works in color and black/white

## Image Optimization

Reduce file sizes without losing quality:

**Online Tools (Free):**
- **TinyPNG** - https://tinypng.com/ (PNG compression)
- **Compressor.io** - https://compressor.io/ (Multiple formats)
- **Squoosh** - https://squoosh.app/ (Google's tool)

**Command Line (Advanced):**
```bash
# Install ImageMagick
brew install imagemagick  # macOS
sudo apt install imagemagick  # Linux

# Resize and optimize
convert input.png -resize 1270x760 -quality 90 output.png

# Batch process all screenshots
for file in *.png; do
  convert "$file" -quality 90 "optimized-$file"
done
```

**Goal:** Keep files under these sizes:
- Logos: <200KB
- Screenshots: <500KB each
- Cover images: <300KB
