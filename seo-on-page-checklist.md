# SEO On-Page Optimization Checklist

## Overview
Comprehensive checklist for optimizing every content piece before publishing. Follow this for all 1,600+ pieces to ensure consistent, search-friendly optimization.

---

## Pre-Publishing Checklist (Complete for EVERY Piece)

### ✅ 1. Title Tag Optimization

**Requirements:**
- [ ] 50-60 characters (max 60)
- [ ] Target keyword in first 50% of title
- [ ] Include power words when relevant (Guide, Complete, 2025, Ultimate, etc.)
- [ ] Unique across entire site (no duplicates)
- [ ] Match user intent (informational, transactional, navigational)

**Formula Templates:**

**Career Guides:**
```
[MOS Code] [Job Title]: Civilian Career Guide 2025
```
Example: `68W Combat Medic: Civilian Career & Salary Guide 2025` (57 chars)

**State Benefits:**
```
[State] Veteran Benefits 2025: Complete Guide
```
Example: `California Veteran Benefits 2025: Complete Guide` (51 chars)

**VA Claims:**
```
[Topic]: Complete Guide for Veterans 2025
```
Example: `VA Disability Nexus Letter: Complete Guide 2025` (52 chars)

**Transition Content:**
```
[Topic]: [Descriptor] Guide for Veterans 2025
```
Example: `Military Resume: Complete Writing Guide 2025` (48 chars)

**⚠️ Common Mistakes:**
- ❌ Too long (truncated in SERPs)
- ❌ No target keyword
- ❌ Generic titles ("Career Guide" instead of "68W Combat Medic Career Guide")
- ❌ Missing year for time-sensitive content

### ✅ 2. Meta Description Optimization

**Requirements:**
- [ ] 150-160 characters (max 160)
- [ ] Include target keyword naturally
- [ ] Include call-to-action (CTA)
- [ ] Describe value/benefit
- [ ] Unique across entire site

**Formula Templates:**

**Career Guides:**
```
Transitioning from [MOS]? Discover civilian career paths, salary ranges ($X-$Y), required certifications, and top employers hiring veterans. Start your transition today.
```

**State Benefits:**
```
Complete guide to [State] veteran benefits: property tax exemptions, education waivers, healthcare access, and employment resources. Save $X,XXX+ annually.
```

**VA Claims:**
```
Step-by-step guide to [VA topic]. Learn requirements, timelines, common mistakes, and how to maximize your rating. Free templates included.
```

**CTAs to Use:**
- "Start your transition today"
- "Get your free checklist"
- "Calculate your benefits"
- "Download free template"
- "Learn the exact steps"

**✅ Good Example:**
```
Transitioning from 68W Combat Medic? Find civilian healthcare jobs, EMT certification paths, salary data ($45K-$85K), and veteran hiring programs. Start today.
```
(159 characters, includes keyword, salary data, CTA)

**❌ Bad Example:**
```
This article discusses career options for combat medics leaving the military and entering civilian employment.
```
(112 characters, generic, no CTA, no compelling benefit)

### ✅ 3. URL Structure

**Requirements:**
- [ ] Clean, readable, hyphenated
- [ ] Include target keyword
- [ ] Lowercase only
- [ ] No special characters (except hyphens)
- [ ] Maximum 3-5 words
- [ ] No dates in URL (except for time-sensitive content)

**URL Patterns:**

**Career Guides:**
```
/career-guides/[branch]/[mos-code]-[job-title]
```
Examples:
- `/career-guides/army/68w-combat-medic`
- `/career-guides/navy/hm-hospital-corpsman`
- `/career-guides/marines/0311-rifleman`

**State Benefits:**
```
/veteran-benefits/[state-name]
```
Examples:
- `/veteran-benefits/california`
- `/veteran-benefits/texas`

**VA Claims:**
```
/va-claims/[topic]
```
Examples:
- `/va-claims/nexus-letter-guide`
- `/va-claims/secondary-conditions`

**Transition Content:**
```
/resources/[topic]
```
Examples:
- `/resources/military-resume-guide`
- `/resources/terminal-leave-calculator`

**Blog Posts:**
```
/blog/[topic-keyword]
```
Example:
- `/blog/skillbridge-programs-2025`

**⚠️ Avoid:**
- ❌ `/article?id=12345`
- ❌ `/career-guides/article-about-68w-combat-medics-transitioning-to-civilian-healthcare-careers`
- ❌ `/page.php?cat=army&id=68w`
- ❌ `/2025/01/15/68w-guide` (dates age poorly)

### ✅ 6. Keyword Optimization

**Requirements:**
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in H1
- [ ] Primary keyword 2-4% density (natural, not stuffed)
- [ ] Secondary keywords (3-5) used naturally throughout
- [ ] LSI keywords included (related terms)
- [ ] Keyword in image alt text (at least 1)
- [ ] Keyword in URL

**Keyword Placement Priority:**
1. Title tag
2. H1
3. First paragraph (ideally first sentence)
4. At least one H2
5. Image alt text
6. URL
7. Meta description
8. Naturally throughout body content

**Example - Primary Keyword: "68W civilian jobs"**

```markdown
# 68W Combat Medic to Civilian Career: Complete Transition Guide

Transitioning from **68W Combat Medic to civilian jobs**? Former Army medics
have dozens of high-paying healthcare career options...

## Civilian Career Paths for 68W Combat Medics

### Emergency Medical Technician (EMT)
**68W to EMT** is one of the most common transitions...

[Continue using variations: "68W civilian careers", "combat medic jobs",
"68W veteran employment", etc.]
```

**⚠️ Avoid Keyword Stuffing:**

❌ **Bad:**
```
68W civilian jobs are important. Many 68W civilian jobs exist. If you want 68W
civilian jobs, you should know about 68W civilian jobs opportunities.
```

✅ **Good:**
```
Transitioning from 68W Combat Medic to civilian healthcare? You have multiple
career paths: EMT, paramedic, registered nurse, physician assistant, and more.
Former combat medics are highly valued in civilian medicine.
```

### ✅ 7. Content Quality Standards

**Requirements:**
- [ ] Minimum 1,500 words (career guides: 2,000+)
- [ ] Original content (no duplicate text)
- [ ] Readable (Grade 8-10 reading level)
- [ ] Scannable (short paragraphs, bullet points, bold text)
- [ ] Actionable (specific steps, not generic advice)
- [ ] Accurate (fact-checked, current data)
- [ ] Updated dates (2025 where relevant)

**Content Structure Formula:**

1. **BLUF (Bottom Line Up Front)** - 100-150 words
   - Answer the main question immediately
   - Key statistics or facts
   - What reader will learn

2. **Body Content** - 1,300-1,800+ words
   - Organized by H2 sections
   - Specific, actionable information
   - Real examples and data
   - Lists and tables

3. **CTA (Call to Action)** - 50-100 words
   - Next steps
   - Links to related resources
   - Tools or downloads

**Reading Level Tools:**
- Hemingway Editor (aim for Grade 8-9)
- Grammarly readability score
- Microsoft Word readability statistics

### ✅ 8. Internal Linking

**Requirements:**
- [ ] 3-5 internal links per page minimum
- [ ] Descriptive anchor text (no "click here")
- [ ] Links to relevant, related content
- [ ] Mix of contextual and navigational links
- [ ] No broken links

**Internal Linking Strategy by Content Type:**

**Career Guides Should Link To:**
1. Related MOS guides (same branch)
2. State benefits guide (for their target relocation state)
3. Resume builder tool
4. VA claims guide (if relevant)
5. Certification guides

**Example:**
```markdown
After completing your certification, explore [California veteran benefits]
(/veteran-benefits/california) if you're relocating to the West Coast. Former
68W medics also transition well to [31K positions](/career-guides/army/31k-dog-handler)
or can use their [GI Bill benefits](/resources/gi-bill-guide) for nursing school.
```

**State Benefits Should Link To:**
1. Career guides for that state's major industries
2. VA claims guides
3. Other nearby state guides (regional comparison)
4. State-specific tools (calculators, job boards)

**VA Claims Guides Should Link To:**
1. Related VA claims guides
2. Specific condition guides
3. Calculator tools
4. Appeal guides

**Alt Text Formula:**

```

```

**✅ Good Examples:**
- `68W Combat Medic transitioning to civilian EMT career`
- `California veteran benefits property tax exemption form`

**❌ Bad Examples:**
- `image1` (not descriptive)

- `` (empty alt text)

- Licensed stock photos (veterans, workplaces)
- Infographics (create original)
- Charts and graphs (original data)

### ✅ 11. Featured Snippet Optimization

**Requirements:**
- [ ] Target question-based query
- [ ] Direct answer in first 40-60 words
- [ ] Formatted for snippet (paragraph, list, or table)
- [ ] Include H2 with question format

**Snippet-Friendly Formats:**

**Paragraph Snippet (Definition):**
```markdown
## What is a VA Nexus Letter?

A VA nexus letter is a medical opinion from a healthcare provider that
establishes a connection ("nexus") between your current disability and your
military service. The letter must include your diagnosis, the provider's
credentials, and a clear statement linking the condition to service.
```

**List Snippet (Steps):**
```markdown
## How to File a VA Disability Claim

1. Gather your medical records (service treatment records)
2. List all conditions to claim
3. Register at VA.gov or eBenefits
4. Complete VA Form 21-526EZ
5. Upload supporting documents
6. Submit your claim
7. Attend C&P exams
8. Wait for VA decision (90-180 days)
```

## Batch Optimization Workflow (For Solo Founder)

**Time-Saving Process:**

1. **Template Creation** (1-time setup)
   - Create Word/Google Doc templates for each content type
   - Pre-fill standard sections (headers, CTAs)
   - Save time on repetitive structure

2. **Keyword Assignment** (bulk process)
   - Assign primary + secondary keywords to all 1,600 pieces at once
   - Use spreadsheet to track

3. **Title/Meta Batch Writing** (dedicated session)
   - Write all titles for Tier 1 content in one sitting
   - Write all meta descriptions for Tier 1 in one sitting
   - Maintain consistency

4. **Schema Generation** (automated)
   - Use schema generator tools
   - Create templates for each content type
   - Copy/paste/modify

5. **Image Optimization** (batch process)
   - Optimize all images at once (bulk tools)
   - Rename files in batch
   - Generate alt text in spreadsheet, then copy
